BIDS Manager

Raw-to-BIDS without custom scripts.

BIDS Manager brings raw-to-BIDS conversion for MRI, EEG, MEG, and physiological recordings into a single integrated workflow. Its intuitive graphical user interface supports interactive metadata inspection, dataset curation, conversion, and validation while orchestrating powerful backend data-management algorithms and conversion engines.

Make raw-to-BIDS less painful.

Less fragmented, less error-prone, more transparent. One tool for the whole BIDS workflow, available as a GUI or a CLI.

One workflow for MRI, EEG, MEG, and physio.

BIDS Manager wires the trusted converters (dcm2niix, mne-bids, bidsphysio) into one orchestrated pipeline. No tool-switching, no per-modality commands, no separate installs.

Scans data and metadata inside every file.

Builds a complete picture of each acquisition, study, and dataset by reading DICOM headers, EEG / MEG file metadata, and sidecars. Most converters only read filenames. BIDS Manager looks inside.

Curate before you convert.

Review every acquisition in an interactive table. Filter out localizers, calibrations, and failed runs. Preview the exact BIDS tree before anything is written to disk.

Automatic metadata enrichment.

After conversion, an enrichment algorithm fills required sidecar fields from the scanned metadata and the BIDS schema. Every decision is logged, so you can review or override it.

Built on the official BIDS schema.

Naming, structure, and validation come from the official BIDS schema, not handwritten rules. The same source of truth drives classification, file naming, and the built-in validator.

Install in one click.

Bootstrap installers for macOS, Linux, and Windows set up a portable Python environment and register a native app launcher. No setup required. Python users can install from PyPI instead.
